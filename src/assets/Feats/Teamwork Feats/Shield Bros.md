
<span style="color:rgb(103, 254, 120)">Requirements</span> -

Gain +2 [[AC]] when you are within the [[Reach|natural reach]] of an ally that wields a shield. This can increase up to +4 ac if 4 shield bros are involved. This basically functions as and does not stack with the [[Block]] action.

Due to how spaces work on grids, having groups of 4 adjacent to each other can be difficult. So for groups, as long as each member is adjacent to another member, that should suffice for gaining this feats benefits.
