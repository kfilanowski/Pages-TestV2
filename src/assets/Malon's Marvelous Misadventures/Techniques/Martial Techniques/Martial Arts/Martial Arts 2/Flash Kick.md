Tier 2 [[Martial Arts]] [[Technique]]
[[Action-Attack]], [[Recharge]]

| Effect      |              |
| ----------- | ------------ |
| Range       | Reach        |
| Target/Area | One Creature |
| Save        | -            |
| Duration    | -            |
With a kick and a flip, perform an [[Unarmed]] [[Attack]] that, on hit, launches the target 10ft in the air above you (this distance increases to match your height - so for most humanoids, this is roughly 15 ft). They have -2 [[AC]] while airborne.
They fall at the end of your turn.
