Tier 3 [[Martial Arts]] [[Technique]]
[[Action-Attack]], [[At-Will]]

| Effect      |                       |
| ----------- | --------------------- |
| Range       | Reach                 |
| Target/Area | One Airborne Creature |
| Save        | -                     |
| Duration    | One Round             |
Make an [[Unarmed]] [[Attack]] against an airborne creature. On hit, it is [[Grounded]], and is instantly pushed 60 ft down.
If it impacts a solid surface, it receives 3d8 bonus damage and is [[Knocked Down]].

[[Upcast]] 1 - +d8 Damage.
