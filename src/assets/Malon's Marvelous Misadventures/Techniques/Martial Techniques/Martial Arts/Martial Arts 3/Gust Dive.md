Tier 3 [[Martial Arts]] [[Technique]]
[[Swift Action]], [[At-Will]]

| Effect      |                               |
| ----------- | ----------------------------- |
| Range       | 1 Mile                        |
| Target/Area | 10 ft Radius, Centered on You |
| Save        | Reflex                        |
| Duration    | -                             |
While airborne, instantly move 1 Mile downwards. If you encounter a solid surface, you cause a gust of wind to push back nearby creatures 10 Ft away and become [[Clumsy]] 1 (Reflex negates).

This ability does not inflict [[Fall Damage]] on you.
