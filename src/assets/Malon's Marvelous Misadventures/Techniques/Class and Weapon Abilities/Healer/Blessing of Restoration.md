[[Healer]] [[Innate Technique]]
[[Passive]], [[Standard Action]], [[At-Will]]

| Effect      |              |
| ----------- | ------------ |
| Range       | Reach        |
| Target/Area | One Creature |
| Save        | -            |
| Duration    | -            |
You recover from all stacks of [[Drained]] every time you rest, regardless of conditions.

As a [[Standard Action]], you can touch a creature to transfer a stack of Drained from them to yourself. A creature can only be affected by this once each week.
