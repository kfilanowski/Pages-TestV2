[[Class]] [[Innate Technique]]
[[Standard Action]], [[Recharge]]

| Effect      |                      |
| ----------- | -------------------- |
| Range       | Reach                |
| Target/Area | One Creature or Item |
| Save        | -                    |
| Duration    | -                    |
On an ally, one harmful [[Spell]] or Supernatural effect immediately ends, unless it is permanent. On an enemy, this functions as [[Dispel]]. For cursed items, this allows the safe removal of the item, but requires a [[CL]] check.
