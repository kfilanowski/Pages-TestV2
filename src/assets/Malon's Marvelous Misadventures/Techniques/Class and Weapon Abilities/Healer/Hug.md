[[Healer]] [[Innate Technique]]
[[Passive]], [[Standard Action]], [[At-Will]]

| Effect      |              |
| ----------- | ------------ |
| Range       | Reach        |
| Target/Area | One Creature |
| Save        | -            |
| Duration    | One Round    |
When you initiate a [[Grab]] on an enemy, they are [[Pacified]] for 1 round instead of the usual effects.

You can [[Mana Burn]] 1 to hug an ally as a Standard Action, putting them in [[Stasis]] for one round, and removing any [[Fear]] effects.
