[[Rogue]] [[Innate Technique]]
[[Immediate Action]], [[Recharge]]

| Effect      |          |
| ----------- | -------- |
| Range       | -        |
| Target/Area | Self     |
| Save        | -        |
| Duration    | 5 Rounds |
Gain the ability to use [[Stealth]] anywhere and [[Hide]] on the same action, as well as cause all [[ACC|Accuracy]] checks against you to fail. Lasts five turns, or until you [[Attack]], cast a [[Spell]], or otherwise take an offensive action. Creatures that are not using [[Detect]] do not get additional checks to spot you.
