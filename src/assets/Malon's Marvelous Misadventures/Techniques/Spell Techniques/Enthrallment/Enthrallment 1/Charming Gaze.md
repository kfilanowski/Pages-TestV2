Tier 1 [[Enthrallment]] [[Spell]]
[[Standard Action]], [[At-Will]]
[[Subtle]], [[Mind-Affecting]]
[[Illusion]]

| Effect      |                     |
| ----------- | ------------------- |
| Range       | 30                  |
| Target/Area | One Creature        |
| Save        | Will                |
| Duration    | 1 Minute or 8 Hours |
You lock your eyes onto a target and [[Charmed|Charm]] them ([[Will]] negates). If you're currently engaged in combat with the creature, the spell's duration is 1 minute. If you are not, the spell's duration is 8 hours. Entering combat against them shortens the remaining duration to 1 minute.
