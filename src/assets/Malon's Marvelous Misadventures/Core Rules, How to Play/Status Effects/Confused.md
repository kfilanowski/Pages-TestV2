Creature considers all others enemies. Every turn, they roll d10+[[LUK]] to determine how they act. For confusion effects that last more than 1 round, each subsequent d10 roll gets 2 added to the result.

| Roll | Effect                                                                   |
| ---- | ------------------------------------------------------------------------ |
| 1-2  | Attack nearest non-[[Companion]] creature once.                          |
| 3-4  | Harm yourself for 2d8+[[STR]] Crush [[Damage]] as a [[Standard Action]]. |
| 5-6  | Consume all movement to wander 15 feet in a random direction.            |
| 7-8  | Babble incoherently as a Standard Action.                                |
| 9-10 | Confusion ends.                                                          |
