# <span style="color:rgb(103, 254, 120)">Uncommon languages</span>

| Morse Code    | A binary code of two sounds mixed to create letters in an alphabet. Communication in Morse code is very slow and impractical, but still serves a useful function for subtle communication.<br> |
| ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Sign Language | An entirely visual language that consists of hand gestures that translates directly to common.<br>                                                                                             |
| Necril        | Language of undead<br>                                                                                                                                                                         |
# <span style="color:rgb(103, 254, 120)"><span style="color:rgb(103, 254, 120)">Secret languages</span></span>

| Druidic        | Typically only passed down from teacher to student with specialized training in nature magic.                                                                                                                                                    |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Thieves Cant   | The language of thieves and criminals. It's more like speaking in code than another language entirely.<br>                                                                                                                                       |
| Semaphore      | A code that uses the position of flags to create letters in an alphabet. Communication and semaphore is very slow and impractical, but still serves a useful function for subtle communication.<br>                                              |
| Bronze         | A unique language that is only understood by creatures with negative intelligence. Typically only spoken by barbarians.<br>                                                                                                                      |
| Fairy Flare    | A language that is only used by a specific type of fairies that cause their bioluminescent antennae to flash in specific patterns. Even if you know the language, it cannot be spoken without the proper biology/equipment, only understood.<br> |
| Ancient Dragon | A particularly old dialect of draconic, spoken only by particularly old creatures that lived during [[The Age of Dragons]].                                                                                                                      |
