- Cannot [[Core Rules, How to Play/Actions/Common Actions/Full Round Actions/Charge]] or [[Dash]]. 
- [[Jump]] distance is halved (minimum 5). This is applies after all other jump modifiers (like [[Enhanced Jump]])
- -4 [[+Athletics]], [[+Perception]] and [[+Acrobatics]]. 
Removed after 10 minutes of inactivity.

Doing anything that causes you to become fatigued again while already fatigued instead causes you to become [[Exhausted]].



<span style="color:rgb(192, 0, 0)">Note - </span>
Because fatigue is relatively easy to remove, it is intended to be a temporary status effect. If you want to start racking up debuffs for arduous journeys, see [[Exhausted]].
