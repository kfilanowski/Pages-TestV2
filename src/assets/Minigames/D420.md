Not really an official game mechanic I expect people to use, but more something fun that my friends and I do. At the start of each session, we essentially make a wish, usually giving our characters some esoteric power or causing an object or creature to materialize into being. We then use Rolz (I'm sure any electronic rolling service will suffice) to roll a d420.

On a <span style="color:rgb(103, 254, 120)">natural 420</span>, it happens! On a <span style="color:rgb(192, 0, 0)">natural 1</span> however, the opposite happens.

The GM is still the final arbiter of how these rolls get implemented. Things that result in direct power increases (teehee I wanna be level 20), should be shamed. This roll is purely for whacky fun and memes. It's a fun way to let give the players a little taste of [[GM Magic]] and occasionally test the [[GM]]'s improv skills.

Because the odds are so low of getting a result (1/210), these kinds of events typically only happen a few times a campaign, if at all.
