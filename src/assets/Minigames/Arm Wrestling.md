Arm wrestling contests are tests of strength that utilize the athletics skill. Each "round" of an arm wrestling contest is effectively like a round in combat for the purposes of determining how long single round buffs last.

Each round calls for a contested athletics check, however you start off with lower dice, progressing to higher dice each round with the following progression.

| Round        | Dice Rolled |
| ------------ | ----------- |
| 1            | d6          |
| 2            | d8          |
| 3            | d10         |
| 4            | d12         |
| 5 and onward | d20         |
Subtract the lower roll from the higher roll. That is how many points the winner of that round gets. The loser of each round gains no points. Once one creature hits 20 points, they win.
