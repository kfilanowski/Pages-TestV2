Level 4 [[Protector]]
Tags: [[Su]]
Action: Swift

| Range          | 30 ft      |
| -------------- | ---------- |
| Target/[[AoE]] | 1 creature |
| Duration       | -          |
| Save           | Will       |
You cause a creature to internally struggle when harming creatures (will negates). Whenever it deals damage, it deals half damage. Whenever it inflicts a harmful effect that doesn't deal damage, the DC is decreased by 4.

The caster also suffers this debuff while the technique persists.

This effect ends if the creature spends 1 round doing nothing.
