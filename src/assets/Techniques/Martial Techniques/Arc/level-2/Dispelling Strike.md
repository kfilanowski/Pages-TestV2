Roll an [[attack roll]] in place of a [[Dispel]] check against a magical effect you can attack. If it is attached to a creature, the attack only effects the spell, not the creature.

If you successfully dispel the effect, the caster takes 2d8+lvl (9+lvl) force damage.

<span style="color:rgb(103, 254, 120)">Upcast</span> - For each additional mana spent, increase damage by d8 (4).
