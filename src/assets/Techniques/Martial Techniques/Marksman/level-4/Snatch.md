Level 4 [[Marksman]]
Tags: [[Su]]
Action: Immediate

| Range          | -                                                    |
| -------------- | ---------------------------------------------------- |
| Target/[[AoE]] | 1 creature targeting you with a ranged weapon attack |
| Duration       | -                                                    |
| Save           | -                                                    |
When attacked by a ranged projectile weapon, roll a [[Reflex]] save against the creatures [[DC]]. If you succeed, you negate the attack entirely. If you are wielding a ranged weapon, transform the projectile into something that fits it and then fire/throw it back as an instant action.

Particularly odd weapons (like the [[Ray Gun]]) require a near exact matching weapon to fire back.
