Action: Standard

| Range          | Reach    |
| -------------- | -------- |
| Target/[[AoE]] | 10x10 ft |
| Duration       | -        |
| Save           | -        |
You create a wall, floor or ceiling of any type that can appear to be made of anything. It blends seamlessly with the nearby terrain to appear as natural as possible. If you cast a new illusory wall, the previous one disappears. 

If the wall would take any damage, it quickly fades. If a creature attempts to move through the wall or otherwise interact with it meaningfully, it quickly fades.

<span style="color:rgb(103, 254, 120)">Upcast</span> - If your [[Caster Level|CL]] is at least 12...
Previous illusory walls do not disappear when a new one is cast. You cannot chain cast illusory walls to make a bigger wall.
