Level 3 [[Cryomancy]]
Tags: [[Frost]]
Action: Standard

| Range          | 60 ft      |
| -------------- | ---------- |
| Target/[[AoE]] | 1 creature |
| Duration       | -          |
| Save           | -          |
Fire a ray of ice at a foe. Make a [[Ranged Attack|ranged]] [[spell attack]]. It deals 4d8+CL (18+CL) cold damage and makes them [[frosty]] and [[entangled]] on hit. If they are already frosty or entangled, they are instead [[entombed]]. Creature can [[Breakout]] as a swift action to escape entangle, or as a standard action if entombed.
