Level 1 [[Clairvoyance (T)]]
Tags: [[Zap]], [[Concentration]], [[Persistent]]
Action: Standard

| Range          | 60 ft                |
| -------------- | -------------------- |
| Target/[[AoE]] | 1 creature or object |
| Duration       | 3 rounds             |
| Save           | -                    |
You cause a dark cloud to appear above the target and it immediately shocks the target with a bolt of lightning for 2d8+CL (9+CL) zap damage (reflex halves). At the start of your next 2 turns, it moves up to 15 feet towards the same target again. Any enemy in a space underneath the cloud during this movement takes the same damage (reflex halves).

If you spend a standard action concentrating on the spell, you can change the target, or manually move it around 15 ft. This also extends the duration by 1 round.

If the current target is [[defeated]], the creature takes [[total cover]] or otherwise leaves the battle, the cloud will instead move to target the nearest enemy.
