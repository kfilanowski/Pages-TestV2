Level 2 [[Psionics]]
Tags: [[Mind-Affecting]], [[Concentration]]
Action: Standard

| Range          | Reach      |
| -------------- | ---------- |
| Target/[[AoE]] | 1 creature |
| Duration       | -          |
| Save           | Will       |
With a touch, you can tap into your targets memories (will negates) or your own, decided when you initiate mental contact. In any case, both parties see what is being viewed. The memory is limited to one minute duration. If gazing into another creatures memories, you must specify the type of memory or specific memory you want to look at.

Against a [[helpless]] creature, this has no save. If you cannot maintain physical contact, the spell ends.
