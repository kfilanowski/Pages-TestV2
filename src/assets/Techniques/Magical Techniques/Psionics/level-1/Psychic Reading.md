Level 1 [[Psionics]]
Tags: [[Mind-Affecting]], [[Subtle]], [[Concentration]]
Action: Standard

| Range          | 30 ft      |
| -------------- | ---------- |
| Target/[[AoE]] | 1 creature |
| Duration       | 3 rounds   |
| Save           | -          |
You focus on a creature's aura to learn more information about them. Roll a [[Caster Level|CL]] check, DC 10+(creature's [[LvL]]). If you succeed, you can learn information from their aura, and even more by concentrating. Every round you learn a piece of information about it from the following list (caster's choice).
* Sexual Orientation
* Race/Species
* Gender
* Level
* Native Language
* What city/sanctuary they consider "home"
* Age
You can only cast this spell on a given creature once per day.
