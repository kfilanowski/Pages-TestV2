Level 2 [[Corruption]]
Tags: [[Concentration]]
Action: Standard

| Range          | 60 ft      |
| -------------- | ---------- |
| Target/[[AoE]] | 1 creature |
| Duration       | 3 rounds   |
| Save           | Fort       |
Cause a creature to atrophy over 3 rounds. This has no immediate effect, but at the start of the creatures turn it withers (fort negates). The first failure fatigues the creature and reduces their speed by 15. The second and third failure reduce their speed by an additional 15.

This effect continues for three turns, regardless if the creature passes its save (unless it has [[Evasion|Stalwart]]).
