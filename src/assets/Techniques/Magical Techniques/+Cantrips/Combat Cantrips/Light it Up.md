[[+Cantrips]]
Tags: [[Fire]], [[Light]]
Action: Standard

| Range          | Reach      |
| -------------- | ---------- |
| Target/[[AoE]] | 1 creature |
| Duration       | 1 minute   |
| Save           | -          |
A mote of flame manifests in an empty hand, or attached to a staff/seal. As part of casting the spell, you can blow this flame onto a creature within reach by making a [[Melee Attack|melee]] [[Spell Attack]]. Does d10+CL (5+CL) fire damage. [[On-hit]] the creature is made [[Glowing]] for 1 round.

By adding the [[Concentration]] tag, you can hold the flame, using it to gently warm things, start fires and provide light (as per [[Torch]]).
