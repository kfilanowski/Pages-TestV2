Tags: -

| Range          | Reach                |
| -------------- | -------------------- |
| Target/[[AoE]] | 1 creature or object |
| Duration       | -                    |
| Save           | -                    |
You touch a surface, including one on a creature, and inscribe a symbol or a single word. The mark must fit a in 4x4 inch square.

In order to use the spell on a creature, it must be willing, you must land a [[Melee Attack]] or you must make a successful [[+sleight of hand]] check.

[[Dispel magic]] automatically succeeds on an arcane mark.
