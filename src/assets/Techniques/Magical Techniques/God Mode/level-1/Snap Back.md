Tags: [[Elusive]]

| Range          | 30 ft      |
| -------------- | ---------- |
| Target/[[AoE]] | 1 creature |
| Duration       | -          |
| Save           | Will       |
Reverse time ever so slightly, only doing so for 1 particular creature. They move back to the position they were in at the start of their last turn. If used on an enemy (will negates).

If you are in combat, it has no effect on a creature that has not acted in combat.
