Level 3 [[Prismancy]]
Tags: [[Fire]], [[Upcast]]
Action: Standard

| Range          | 30 ft      |
| -------------- | ---------- |
| Target/[[AoE]] | 1 creature |
| Duration       | -          |
| Save           | -          |

Fire a beam that magnetizes to [[Glowing]] creatures. Make a [[Ranged Attack]]. The target takes 4d8+CL (18+CL) fire damage. If they are not glowing, they begin glowing. If they already are glowing, this attack has [[Seeking]] and deals an additional 10 damage. This removes the glowing.

<span style="color:rgb(103, 254, 120)">Upcast </span>- Every 2 [[Caster Level|CL]]s, increase the spells range by 5 ft.
