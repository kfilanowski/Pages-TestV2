Level 3 [[Prismancy]]
Tags: [[Light]]
Actions: Standard

| Range          | Reach                |
| -------------- | -------------------- |
| Target/[[AoE]] | 1 creature or object |
| Duration       | Until Rest           |
| Save           | -                    |
You cause the target of the spell to emanate an exceptionally bright light. The area within 60 ft of the target is bathed in bright light.

You can use the spell effectively as [[dispel magic]] against a spell with the [[dark]] tag. If you succeed, the light spell consumes the dark.
