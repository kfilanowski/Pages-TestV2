Separate from [[Carry Weight]], which determines raw lifting ability, inventory represents how much stuff you can carry on your person. These represented by <span style="color:rgb(192, 0, 0)">inventory slots</span>. By default characters have<span style="color:rgb(192, 0, 0)"> 20 </span>of (or more with high [[STR]]).

Other modifiers, such as [[size]] can affect inventory, with each size over medium granting +4 slots and each size under subtracting 4.

Every item from the biggest sword to the tiniest potion has an inventory slot stat, representing how annoying it is to carry around with a minimum of 1. Items do not consume inventory slots when you have them equipped, such as your [[armor]].

I think basic adventuring supplies, such sleeping bags, tents, rations for short journeys and such should not require inventory slots. Any such thing with abnormal qualities or bulk however should.

Common pieces of equipment consume the following number of inventory slots...

| <span style="color:rgb(0, 176, 80)">Equipment</span>   | <span style="color:rgb(0, 176, 80)">Slots</span> |
| ------------------------------------------------------ | ------------------------------------------------ |
| Light/1h Weapons                                       | 1                                                |
| 2h weapons (including most range weapons)              | 2                                                |
| Light Armor                                            | 1                                                |
| Med/Heavy Armor                                        | 2                                                |
| Buckler                                                | 1                                                |
| Shields                                                | 2                                                |
| Most baubles, boots, gloves and headwear               | 1                                                |
| Potions, toxins and other alchemical items             | 1                                                |
| [[Staff Component]]                                    | 1 (or 2 if also a 2h weapon)                     |
| <span style="color:rgb(0, 176, 80)">Other Items</span> |                                                  |
| 1 tiny creature                                        | 1                                                |
| 1 small creature                                       | 2                                                |
| 1 medium creature                                      | ~5                                               |
| 1 large creature                                       | ~20                                              |
| 1 huge creature                                        | ~50                                              |
When denoting items on your character sheet, you can do the following to visually help represent filled inventory slots.

| <span style="color:rgb(0, 176, 80)">Slots</span> | <span style="color:rgb(0, 176, 80)">Item</span> |
| ------------------------------------------------ | ----------------------------------------------- |
| 1                                                | Spare greatsword                                |
| 2                                                | *                                               |
| 3                                                | Spare heavy armor                               |
| 4                                                | *                                               |
| 5                                                | Bola                                            |
| 6                                                | Bola                                            |
| 7                                                | Beeg rock                                       |
| 8                                                | *                                               |
| Etc.                                             | *                                               |
If you prefer physical sheets, doing something similar like drawing a line down the numbered lines for each additional slot an item takes up works too.
